Estudando HTML

Nossos estudos começaram falando a respeito de um item muito sério que é a organização. Isso poruqe a linguagem HTML é responsável apenas pela marcação dos elementos na janela do navegador. O HTML não cria nada, ele manda buscar ou utiliza as letrinhas que você digita

Parágrafosm

Para colocarmos parágrafos, usamos os elementos "p" e "/p". Para fazer uma quebra de linha, mantendo o mesmo parágrafo, usamos "br". Temos ainda parágrafos especiais, denominados títulos, cabeçalhos ou headings em que são usados elementos de "h1" a "h8" e seus respectivos fechamentos "/h1" a "/h8".

Imagens

Uma imagem aparece na janela do navegador pelo uso do elemento "img". Esse elemento possui o atributo "scr", responsável pela localização da imagem. Para trabalhar melhor com a imagem, você pode colocá-la dentro das tags de marcação de parágrafo.
